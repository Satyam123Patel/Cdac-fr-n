## 📋 Course Syllabus

### Topics Covered (29 Sessions)

| Session | Topics |
|---------|--------|
| **1-2** | JDBC, Connection, Statement, PreparedStatement, Transactions |
| **3** | J2EE Architecture, Containers, WAR/EAR Deployment |
| **4-7** | Servlets Lifecycle, HttpServlet, Session Management, Filters |
| **8-9** | JSP Syntax, Directives, EL, JSTL, MVC Pattern |
| **10-13** | Hibernate ORM, Entity Mapping, HQL, Relationships |
| **14-17** | Spring Core, IoC, DI, Beans, Spring MVC |
| **18-20** | Spring Boot, Auto-configuration, Starters, Embedded Server |
| **21-22** | Spring Data JPA, Repositories, Query Methods |
| **23** | Spring AOP, Aspects, Pointcuts, Advice Types |
| **24-25** | REST Services, HTTP Methods, RestTemplate |
| **26-27** | Unit Testing, MockMvc, Spring Security, JWT |
| **28-29** | Microservices, API Gateway, Service Discovery, Circuit Breaker |

### Learning Objectives

- Build enterprise web applications using Java EE technologies
- Master Servlets, JSP, and MVC architecture
- Implement data persistence with Hibernate and Spring Data JPA
- Create RESTful APIs with Spring Boot
- Secure applications with Spring Security and JWT
- Design and deploy microservices architecture
