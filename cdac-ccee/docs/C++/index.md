## 📋 Quick Reference

### Key Concepts Summary

| Topic | Key Points |
|-------|------------|
| **OOP Pillars** | Encapsulation, Abstraction, Inheritance, Polymorphism |
| **Access** | private (class only), protected (+ derived), public (all) |
| **Constructors** | Default, Parameterized, Copy; same name as class |
| **Inheritance** | Single, Multiple, Multilevel, Hierarchical, Hybrid |
| **Polymorphism** | Compile-time (overloading), Runtime (virtual) |
| **Memory** | `new`/`delete` (calls constructor), `malloc`/`free` (doesn't) |
| **STL** | vector, stack, queue, map, set |
