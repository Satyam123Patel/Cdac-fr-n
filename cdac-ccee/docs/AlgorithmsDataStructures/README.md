   - Definitions

## 🔥 Quick Revision Checklist

### Before Exam (Last 24 Hours)

**Must Review:**
- [ ] Big O complexity chart
- [ ] Stack vs Queue
- [ ] Array vs Linked List
- [ ] Tree traversal outputs
- [ ] BST properties
- [ ] Sorting algorithm complexities
- [ ] Hash collision resolution
- [ ] BFS vs DFS
- [ ] Dijkstra's algorithm
- [ ] Greedy vs DP

**Common Mistakes to Avoid:**
- ❌ Confusing O(n) with O(log n)
- ❌ Forgetting base case in recursion
- ❌ Wrong tree traversal order
- ❌ Mixing up stable/unstable sorts
- ❌ Incorrect graph representation choice
