## 📋 Course Syllabus

### Topics Covered (18 Sessions)

| Session | Topics |
|---------|--------|
| **1** | Introduction to DBMS, Types of DBMS, MySQL Clients |
| **2** | Data Models, ERD, Codd's 12 Rules, SQL Categories, DDL |
| **3-4** | Normalization (1NF-5NF), Functional Dependencies, DML |
| **5** | MySQL Data Types, Constraints, Aggregate Functions, GROUP BY |
| **6** | Relational Algebra, Joins, AUTO_INCREMENT |
| **7** | Subqueries, TCL, DCL, Views |
| **8** | Indexes, ACID Properties, Storage Engines |
| **9** | Stored Procedures, Parameters (IN/OUT/INOUT) |
| **10** | Flow Control (LOOP, WHILE, REPEAT), Conditionals |
| **11** | Loop Constructs, Functions, Built-in Functions |
| **12** | Cursors |
| **13** | Triggers (BEFORE/AFTER, NEW/OLD) |
| **14** | Error Handling and Exceptions |
| **15** | NoSQL Introduction, CAP Theorem, BASE Model |
| **16-18** | MongoDB (CRUD, Operators, Indexing) |

### Learning Objectives

- Understand database fundamentals and RDBMS concepts
- Master SQL for data definition, manipulation, and querying
- Design normalized database schemas
- Implement constraints, views, and indexes
- Work with MongoDB for NoSQL database solutions
- Write stored procedures, functions, and triggers
- Manage transactions and ensure data integrity
