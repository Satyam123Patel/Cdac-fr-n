## 📋 Course Syllabus (PG-DAC Feb 2025)

### Course Overview
**Total Sessions**: 28 | **Duration**: 112 Hours

This module covers the complete Web Programming Technologies curriculum including client-side development, server-side programming with Node.js, and modern frontend frameworks.

### Topics Covered

#### 1. Web Fundamentals (Sessions 1-4)
- Internet architecture, DNS, HTTP/HTTPS protocols
- HTML5 semantic elements, forms, multimedia
- CSS3 selectors, box model, positioning, animations
- Responsive design, media queries, Bootstrap framework

#### 2. JavaScript Core (Sessions 5-12)
- Variables, data types, operators, control structures
- Arrays, strings, dates, objects, functions
- ES6+ features (let/const, arrow functions, classes)
- DOM manipulation, events, form validation
- JSON, jQuery selectors, events, animations
- AJAX, Fetch API, Axios HTTP client

#### 3. Node.js & Express (Sessions 13-18)
- Node.js runtime, event loop, modules
- Asynchronous programming, promises, async/await
- npm, package management, file system
- HTTP server, streams, buffers
- Express.js routing, middleware, templates

#### 4. React & Redux (Sessions 19-28)
- React components, props, JSX
- State management, hooks (useState, useEffect, useRef)
- Custom hooks, context API, error boundaries
- React Router, code splitting
- Redux: actions, reducers, store
- Redux Toolkit, async thunks, selectors

### Learning Outcomes
- Build responsive, accessible web applications
- Create RESTful APIs with Node.js and Express
- Develop dynamic UIs with React
- Implement state management with Redux
- Apply industry best practices and patterns
